💼 AI Salary Predictor

AI Salary Predictor is a machine learning based web application developed using Python and Streamlit that predicts employee salaries based on multiple professional and demographic factors such as Age, Gender, Education Level, Job Title, and Years of Experience.

The project demonstrates a complete end-to-end machine learning workflow including:

Data preprocessing
Feature encoding
Model training
Salary prediction
Data visualization
Explainable AI integration

The application uses the Random Forest Regression algorithm from Scikit-learn to train the prediction model on salary data. User inputs are collected through an interactive Streamlit interface and processed into machine-readable format to generate real-time salary predictions.

To improve transparency and trust in predictions, the project integrates SHAP (SHapley Additive exPlanations), which explains how each feature contributes to the final predicted salary using visualization techniques like waterfall plots.
